ImageBot

The ImageBot is a discord bot which creates beautiful photomosaic images of your profile picture out the profile pictures of all the other users in your chat group.

It is currently servering over 10,000 users in discord

Hover over these images to take a closer look

Commands

• Type "*grab" followed by a couple words to search for a image or the nth image. to search for the nth image
• Type "*photomosaic" and @ (mention) to create a photomosaic of their profile picture.
• Type "*grabpp" and @ (mention) someone to get their profile picture.
• Type "*help" for the help menu
